# Hospital Management System

"A menu-driven Hospital Management System developed in C for managing patient records and appointments."

# Overview

Hospital Management System is a console-based application developed in C to manage patient records and appointments efficiently. The project demonstrates the use of fundamental Data Structures and Programming concepts such as Arrays, Structures, Linked Lists, Dynamic Memory Allocation, and Searching.

# Features

* Register new patients
* Display all registered patients
* Search patients by ID or Name
* Schedule appointments
* View scheduled appointments
* Delete appointments
* Dynamic appointment management using Linked Lists

# Technologies Used

* C Programming
* Arrays
* Structures
* Linked Lists
* Pointers
* Dynamic Memory Allocation (`malloc`, `free`)
* Linear Search

# Project Structure

* Patient records are stored using Arrays and Structures.
* Appointment records are managed using a Linked List for dynamic insertion and deletion.
* Memory is allocated dynamically for appointment nodes to optimize storage usage.
